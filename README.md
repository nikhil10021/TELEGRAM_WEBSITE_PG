# 🚀 DealDrop — Telegram Deals Landing Page

A modern, mobile-responsive Telegram deals landing page built with **React + Vite**.
Deployed on **Vercel**.

[Click Here to View Live Site](#) ← Vercel link yahan daalna deploy ke baad

## 📸 Features
- ⚡ Live countdown timer to next deal
- 🔔 Rotating social proof notifications
- 📱 Fully mobile responsive
- 🎨 Dark theme with amber accent
- 🔗 Multiple Telegram join buttons
- 🖼️ Deal image gallery with hover effects

---

## 🛠️ Tech Stack
- React 18
- Vite
- CSS Custom Properties (no Tailwind)
- React Router DOM
- Deployed on Vercel
